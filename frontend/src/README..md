# CodeAlpha Social Media Platform

## Features

- User Registration
- User Login (JWT)
- Profile Page
- Create Posts
- Upload Images
- Like Posts
- Comment on Posts
- Follow/Unfollow Users
- Delete Posts

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- JWT
- Multer

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
